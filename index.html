Research Interests

Phase Space Calculation

The following C++ function calculates the phase space normalization factor that allows you to convert the relative weights of TGenPhaseSpace into absolute phase space integral weights. (Yes, now you can perform phase space integrals for multiple particles without losing your mind.)

The formula for the phase space normalization factor is:

\[ \text{factor} = \begin{cases} 0 & \text{if } n < 2 \text{ or } E \leq 0 \text{ or } M \leq 0 \\ \frac{2 \pi^{n} (2E)^{n-2}}{(2\pi)^{3(n-1)} M (n-2)!} & \text{if } n \geq 2 \end{cases} \] 


// Calculate the phase space normalization factor
Double_t CalculatePhaseSpaceFactor(Int_t numParticles, Double_t availableEnergy, Double_t parentMass)
{
    if (numParticles < 2 || availableEnergy <= 0.0 || parentMass <= 0.0)
    {
        std::cerr << "Error: Invalid parameters for phase space calculation" << std::endl;
        return 0.0; // Because bad inputs deserve nothing
    }

    double piPower = numParticles;            // Pi factor for each particle (π is everywhere)
    double twoPiCubePower = numParticles - 1; // (2π)³ divisor for each particle
    double teCmTmPower = numParticles - 2;    // availableEnergy factor for intermediate states

    double factor = 2 * pow(TMath::Pi(), piPower) /
                    pow(2.0 * TMath::Pi(), 3 * twoPiCubePower) *
                    pow(2.0 * availableEnergy, teCmTmPower) /
                    parentMass;

    if (numParticles > 2)
    {
        factor /= double(factorial(numParticles - 2)); // Divide by factorial because math says so
    }

    return factor; // Voilà, your phase space factor
}
